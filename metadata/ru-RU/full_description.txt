OpenBible — открытое, простое и лёгкое приложение для Android для чтения Библии без лишнего. Только текст Библии — без заголовков, комментариев, планов чтения и подсказок.

Поддерживает множество языков и переводов Библии (обновляются при проблемах с форматированием). Выбирайте перевод, книгу, главу или ищите стихи по содержимому.

Настройка чтения: цветовые темы, выравнивание текста, разделённый экран, размер шрифта, номера стихов. Настройки берутся из системных по умолчанию.

Изменяйте шрифт щипком или двойным кликом. Выделяйте текст для копирования/поделиться. Все настройки и переводы сохраняются с резервным копированием.
