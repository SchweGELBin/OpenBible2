- Начальная поддержка Android 17
- Показывает уведомление после завершённого резервного копирования
- Исправляет неполные переводы приложений (откат на английский)
- Обновлены переводы (русский, польский)

Улучшенная функциональность поиска:
- Поиск по нескольким запросам, разделённым "+"
- Исключение поиска с помощью "~"
- Префикс "_" игнорирует регистр букв
- Префикс "=" принуждает точный поиск
- По умолчанию — неточный поиск
--> Для простого поиска достаточно ввести слова в нужном порядке
